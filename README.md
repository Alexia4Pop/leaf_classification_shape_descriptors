# Image Processing Project: Leaf Classification

## 1. Project Description
The purpose of this project is to process leaf images and extract geometric shape descriptors to classify them into different species. The program performs image binarization, contour tracing, and region filling using morphological reconstruction. It calculates key features such as Elongation, Circularity, and Solidity, saving the results to a CSV file and visualizing the distribution in a "Feature Space" plot to analyze species separability.

## 2. Technologies Used

* **Libraries:** OpenCV 4.9.0
* **Tools:** Visual Studio 2022
